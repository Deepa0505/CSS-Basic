Defining Color Value

Colors in CSS often specifies in the following formats:

  1. 1.A Color keyword - like - "red" , "green" , "blue" , "transparent" , etc.
  2. 2.A HEXA Value - like "#ff0000" . "#00ff00" , etc
  3. 3.An RGB Values (RED, GREEN, BLUE)- like "rgb(255, 0 , 0)"
